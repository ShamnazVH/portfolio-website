# Portfolio Sharing Guide

## üì¶ How to Share This Portfolio

### **Option 1: Direct File Sharing**
1. **Zip the entire folder** containing:
   - `package.json`
   - `src/` folder
   - `public/` folder
   - All other files

2. **Share the zip file** via:
   - Email
   - Google Drive
   - Dropbox
   - WeTransfer

### **Option 2: GitHub (Recommended)**
1. **Install Git** from [git-scm.com](https://git-scm.com/download/win)
2. **Create account** on [github.com](https://github.com)
3. **Create new repository** on GitHub
4. **Upload files** using GitHub's web interface or Git commands

### **Option 3: Online Code Editors**
1. **CodeSandbox:** Upload to [codesandbox.io](https://codesandbox.io)
2. **CodePen:** Copy code to [codepen.io](https://codepen.io)
3. **Replit:** Upload to [replit.com](https://replit.com)

## üöÄ How to Run This Portfolio

### **Prerequisites:**
- Node.js (Download from [nodejs.org](https://nodejs.org))
- npm (comes with Node.js)

### **Steps:**
1. **Extract/unzip** the portfolio files
2. **Open terminal/command prompt** in the portfolio folder
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm start
   ```
5. **Open browser** to `http://localhost:3000`
